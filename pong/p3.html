Basically, you store a function inside of a variable. Using certain notation, you can make that function "run" at any time you'd like. To create a function in JS: 
    //this is a function named "sayHello"
    var sayHello = function(){

    };
    sayHello(); //will run everything inside: alerts "Hello!"
Inside of a function, you can write any code you'd like. Don't worry about what those parenthesis are for yet. We'll get to it.

A Little Bit of DOM
Get ready for a MASSIVE detour! I want to explain some very important things to you. So far, we've been using alert and prompt to show things to our user. You've probably been wondering how to do input and output in a much nicer way. Nicer ways exist, trust me. The only issue is, this is a beginner tutorial and I'm trying to cover as little as possible at once.

However, right now we need to start displaying text on the screen. Unfortunately, to do this we have to move a little outside of programming, and into the realm of design and webpage structuring. A little bit. Luckily there are some nice ways to have JS interact with those things. But youll have no idea what's going on unless I at least give you a brief explanation.

The file you're working on right now ends in ".html" because it is an HTML file. That's HyperTextMarkupLanguage. A bunch of tags that you write to give structure to a webpage, tell the browser where everything is, for the most part. If we want input boxes, nice user interfaces, and text showing up on the screen, thats how we do it. Through HTML. A quick search will give you more information. But this isn't an HTML tutorial, this is a JS tutorial. But, unfortunately, I'm going to need you to write one line of HTML. After that, we can use some of those built in functions JS has to interact with it. Add this to your file, right above the script tags: 
    <canvas id="game"></canvas>
That, my friend, is called canvas. A nice html element that we can use JS to interact with. It lets us draw things, animate things, write things. It is where our game will exist. the "id" part lets JS know how to find our canvas.
Interacting with Canvas
To start interacting with canvas, we need to use some default functions to first find the canvas, and then let it know we intend to do stuff to it. We will also need to give the canvas a background color (black), so you can see it, and give it a height and width on our page. All the functions and properties you see are built in. Don't worry about them too much for now. 
    var canvas = document.getElementById('game'); //find our canvas
    canvas.style.background = "black"; //background to black
    //we'll need these variables later as well
    var height = 500; //500 pixels
    var width = 900;
    //change height and width
    canvas.height = height; //set height property to our variable
    canvas.width = width;
    //variable we use to control canvas:
    var ctx = canvas.getContext('2d');
You should now see a black box on your page!
